## Instalation

1. Clone this repository

   ```sh
   git clone https://github.com/EPJA-popit/dummy.git
   ```

2. Run npm installation process

   ```sh
   npm i
   ```

3. Run @popit/cli package initialization script
   ```sh
   npx popit-cli --init
   ```
